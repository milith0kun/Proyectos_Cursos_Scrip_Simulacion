Metatags Importantes

  • charset: Define la codificación de caracteres (UTF-8)
  • viewport: Controla la visualización en dispositivos móviles
  • description: Describe la página para motores de búsqueda
  • keywords: Palabras clave para SEO
  • author: Autor del documento

Open Graph Tags

Las etiquetas Open Graph permiten controlar cómo se ve tu página cuando se comparte en redes sociales:

  • og:title: Título para redes sociales
  • og:description: Descripción para redes sociales
  • og:type: Tipo de contenido

Tipos de Iconos

Los iconos son importantes para la identidad visual de tu sitio:

  • Favicon: Icono en la pestaña del navegador (16x16 píxeles)
  • Apple Touch Icon: Para dispositivos iOS
  • Android Icon: Para dispositivos Android
